# Wellysa.com - Main Website

Główne centrum pierwszego kontaktu z marką Wellysa.

## Struktura projektu

- `/app` - Next.js App Router (strony i layouty)
- `/components` - Komponenty React
- `/public` - Statyczne pliki (obrazy, ikony)
- `/lib` - Narzędzia i utilities
- `/styles` - Globalne style

## Uruchomienie

```bash
npm install
npm run dev
```

Strona dostępna pod: http://localhost:3000

## Główne sekcje

1. **Ekran powitalny** - Ludzik (Avatar) + główne menu
2. **Asystent** - Interaktywny chatbot (Intercom)
3. **WIEDZA** - Wikipedia ekosystemu
4. **WYNIKI** - Wizualizacje (wyblurowane)
5. **PROFIL** - Przykładowe dane profilowe
6. **WSPÓŁPRACA** - Formularze dla różnych interesariuszy

## Technologie

- Next.js 14 (App Router)
- React 18
- TypeScript
- Tailwind CSS
- Intercom (Chatbot)
- Framer Motion (animacje)
